🔧 MotherCore Icon Conversion Guide

Transform your AI-generated images into flexible, customizable SVG icons

📐 Step 1: Convert Images to SVG

1 Choose Your Conversion Method:

Adobe Illustrator

Best quality, manual control

Image Trace → Expand → Export SVG

Inkscape (Free)

Open source, good quality

Trace Bitmap → Manual cleanup

Vector Magic

Online converter, automated

Upload → Auto-trace → Download SVG

Figma (Free)

Web-based, easy to use

Import → Vectorize → Export SVG

2 Clean Up Your SVG:

After conversion, you'll need to structure your SVG properly for color customization.

🛠️ Step 2: Structure SVG for Color Control

1 Basic SVG Template Structure:
<svg viewBox="0 0 24 24" fill="none" xmlns="http://www.w3.org/2000/svg"> <!-- Main icon shape (uses primary color) --> <path class="icon-primary" d="M12 2L22 7v10L12 22 2 17V7l10-5z"/> <!-- Accent details (uses accent color) --> <circle class="icon-accent" cx="12" cy="12" r="3"/> <!-- Neural pathways (uses primary with opacity) --> <path class="icon-neural" d="M8 8l8 8M16 8l-8 8" stroke-width="0.5"/> </svg>
2 CSS Class Structure for MotherCore:
/* CSS Variables for MotherCore Colors */ :root { --mothercore-primary: #fefcac; /* Main gold/cream */ --mothercore-accent: #6e3d21; /* Amber-red accent */ --mothercore-bg: #161719; /* Dark background */ } /* Icon Classes */ .icon-primary { fill: var(--mothercore-primary); stroke: var(--mothercore-primary); } .icon-accent { fill: var(--mothercore-accent); stroke: var(--mothercore-accent); } .icon-neural { fill: none; stroke: var(--mothercore-primary); opacity: 0.6; } .icon-background { fill: var(--mothercore-bg); }

⚙️ Step 4: Implementation in Your App

1 React Component Structure:
// IconComponent.tsx import React from 'react'; interface IconProps { name: string; size?: number; className?: string; primaryColor?: string; accentColor?: string; } const Icon: React.FC<IconProps> = ({ name, size = 24, className = '', primaryColor = '#fefcac', accentColor = '#6e3d21' }) => { const iconStyle = { '--mothercore-primary': primaryColor, '--mothercore-accent': accentColor, } as React.CSSProperties; return ( <span className={`mothercore-icon ${className}`} style={iconStyle} > {getIconSVG(name, size)} </span> ); }; const getIconSVG = (name: string, size: number) => { const icons = { close: `<svg width="${size}" height="${size}" viewBox="0 0 24 24">...</svg>`, folder: `<svg width="${size}" height="${size}" viewBox="0 0 24 24">...</svg>`, // ... more icons }; return <div dangerouslySetInnerHTML={{__html: icons[name]}} />; };
2 Dynamic Color Theming:
// Theme System const themes = { default: { primary: '#fefcac', accent: '#6e3d21', background: '#161719' }, matrix: { primary: '#00ff41', accent: '#008f11', background: '#0a0a0a' }, corporate: { primary: '#ffd700', accent: '#ff6b35', background: '#1a1a1a' } }; // Usage in component <Icon name="close" primaryColor={currentTheme.primary} accentColor={currentTheme.accent} />
3 CSS Custom Properties for Global Control:
/* Global CSS Variables */ .app-container { --icon-primary: #fefcac; --icon-accent: #6e3d21; --icon-background: #161719; --icon-neural: rgba(254, 252, 172, 0.6); } /* Dark mode override */ .app-container.dark-mode { --icon-primary: #00ff41; --icon-accent: #ffd700; --icon-background: #0a0a0a; } /* All icons automatically update */ .mothercore-icon .icon-primary { fill: var(--icon-primary); stroke: var(--icon-primary); } .mothercore-icon .icon-accent { fill: var(--icon-accent); stroke: var(--icon-accent); }

/* CSS Animations for Neural Effects */ @keyframes neural-pulse { 0%, 100% { opacity: 0.3; } 50% { opacity: 0.8; } } @keyframes circuit-flow { 0% { stroke-dashoffset: 20; } 100% { stroke-dashoffset: 0; } } .icon-neural { animation: neural-pulse 2s ease-in-out infinite; stroke-dasharray: 2 2; animation: circuit-flow 1s linear infinite; } .mothercore-icon:hover .icon-neural { animation-duration: 0.5s; }

// Dynamic icon states const IconWithState: React.FC<{state: 'normal' | 'active' | 'error'}> = ({ state }) => { const stateColors = { normal: { primary: '#fefcac', accent: '#6e3d21' }, active: { primary: '#00ff41', accent: '#ffd700' }, error: { primary: '#ff4444', accent: '#cc0000' } }; return ( <Icon name="status" primaryColor={stateColors[state].primary} accentColor={stateColors[state].accent} className={`icon-state-${state}`} /> ); };
3 Icon Size Scaling:
// Responsive icon sizing const iconSizes = { xs: 12, sm: 16, md: 24, lg: 32, xl: 48, xxl: 64 }; // Usage <Icon name="folder" size={iconSizes.lg} /> <Icon name="close" size={iconSizes.sm} /> /* CSS for consistent scaling */ .mothercore-icon { display: inline-flex; align-items: center; justify-content: center; } .mothercore-icon svg { width: 100%; height: 100%; }

⚠️ Important Notes:
  • Always use viewBox instead of fixed width/height for scalability
  • Keep SVG files clean - remove unnecessary groups and transforms
  • Use CSS classes instead of inline styles for theme flexibility
  • Test icons at multiple sizes (16px, 24px, 48px) to ensure clarity
  • Maintain consistent stroke-width ratios across all icons
